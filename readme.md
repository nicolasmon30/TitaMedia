# TITA MEDIA PRUEBA HTML

## STARTING 🚀

Estas son las instrucciones para correr la prueba

### prerequisites  📋

Se requieren estos requisitos para poder correr el proyecto

```
- node 10.16.3
- npm 6.9.0
```

### Installation  🔧

```
- npm i
```
or
```
- npm install
```

### Deploy 📦

development (watch)
```
- npm run dev
```

production (minified files)
```
- npm run build
```

### Builded with 🛠️

 - Node Js
 - Webpack
 - SASS
 - JavaScript
